# Pocono Pet Medication Index

Pocono Pet Medication Index is an open reference project for pet owners reviewing veterinary prescription questions with a community pharmacy team. It organizes common medication questions into Markdown notes, checklist pages, and small data files that can be read directly in GitHub or published through GitHub Pages.

The project is connected in style and scope to [Pocono Community Pharmacy](https://www.poconocommunitypharmacy.com/), an independent community pharmacy serving Brodheadsville, East Stroudsburg, and nearby Pocono communities. The primary Pocono source context for this repository is [Pocono pet medication support](https://www.poconocommunitypharmacy.com/medication-information/pet-medication-support). The broader [Pet Care service page](https://www.poconocommunitypharmacy.com/pet-care) is treated as a service/support page, not as the main source for medication reference notes.

This repository is not a veterinary clinic site, a drug catalog, or a substitute for veterinary care. It does not provide numeric amount tables, diagnose animals, approve medication changes, or tell an owner to start, stop, repeat, or change a pet medication. Medication decisions for a pet should be made by the veterinarian who knows the animal’s species, weight, age, diagnosis, medical history, current medications, and monitoring needs.

## Who this project is for

This project is for pet owners and caregivers who:

- have a veterinary prescription and want to ask better questions;
- need to understand what information to have ready before calling a pharmacy;
- are checking refill status, bottle label details, storage notes, or availability;
- manage medication for more than one pet in a household;
- want a clearer boundary between veterinary decisions and pharmacy support.

The pages are written in a practical pharmacy desk-note style. They use short sections, owner checklists, and repeated headings so the reader can scan quickly.

## What this project does

The project helps organize:

- medication names and categories;
- questions for the veterinarian;
- questions for the pharmacy;
- refill and label concerns;
- species, weight, and household safety reminders;
- side-effect watch notes that should not be ignored;
- compounding and travel refill checklists.

## What this project does not do

This project does not replace a veterinarian. It does not provide treatment approval, dosing directions, home-care shortcuts, diagnosis, or medication substitutions. It does not suggest that one pet’s medication can be shared with another pet. It is not a dispensing channel, product catalog, or medication sales page.

## How to use the notes safely

Use the pages as a preparation layer. Read the relevant note, write down the pet-specific details, then call the correct place. If the question is about symptoms, suitability, monitoring, missed dose handling, or whether medication should continue, that belongs with the veterinary office. If the question is about refill status, label wording, storage, availability, or pharmacy communication, the pharmacy can usually help organize the next step.

A good call often starts with: pet name, species, current weight, medication name, bottle label direction, refill number, and what changed. Those details help prevent guessing.
