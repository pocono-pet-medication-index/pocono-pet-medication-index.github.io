# Getting Started with the Pocono Pet Medication Index

## Why this project exists

Pocono Pet Medication Index is a small open reference project for pet owners who are trying to organize medication questions after a veterinary visit, during a refill conversation, or before calling a pharmacy. It is maintained in the style of community pharmacy desk notes: practical, cautious, and focused on the handoff between owner, veterinarian, and pharmacist.

The project connects to the pet prescription support work of [Pocono Community Pharmacy](https://www.poconocommunitypharmacy.com/). It is not a veterinary clinic website and it does not replace veterinary care. It is a way to keep common owner questions in one place: What does the bottle label say? Is the refill number active? Does the veterinary office need to authorize more medication? Does the pet’s current weight or species change the conversation?

## Who should use it

This index is for owners who already have a veterinary prescription, are discussing one with a veterinary office, or are preparing to ask better questions. It can also help caregivers who manage medication for more than one pet in the same household.

It is especially useful when the owner has the bottle nearby but is not sure what to ask next. A label can contain the medication name, pet name, strength, directions, refill number, storage note, and pharmacy contact. Those details are often enough to turn a vague concern into a clear call.

## How to read these notes

Start with the page closest to the question. If the issue is a refill or a confusing label, use the refill page. If the issue is a side effect concern, use the watch-notes page. If the issue is a particular medication, read that medication note and write down the questions that belong with the veterinarian versus the pharmacy.

The medication pages follow the same structure so they are easy to scan. Each page includes where the medication usually comes up, what the pharmacy team wants to clarify first, what to ask the veterinarian, what to ask the pharmacy, and what the page does not do.

## What this project does not do

This project does not diagnose pets, approve medications, provide numeric amount tables, change veterinary instructions, or tell an owner to start or stop a medication. Medication decisions for a pet should be made by the veterinarian who knows the animal’s species, weight, age, diagnosis, medical history, current medications, and monitoring needs.

## A practical pharmacy note

This repository works best when the owner uses it before a call, not instead of a call. Write down the questions, read the bottle label, and decide which questions belong with the veterinary office and which belong with the pharmacy.

That small preparation often makes the conversation calmer and more accurate.
