# Pocono Pet Medication Index

This GitHub Pages site is the public index for the Pocono Pet Medication Index project. It collects owner-friendly reference notes about veterinary prescriptions, refill questions, pharmacy availability, label clarification, and medication safety conversations.

The project is maintained in a community pharmacy support style and is connected to [Pocono Community Pharmacy](https://www.poconocommunitypharmacy.com/). It is not a veterinary clinic site and does not replace veterinary care.

## Boundary note

Medication decisions for a pet should be made by the veterinarian who knows the animal’s species, weight, age, diagnosis, medical history, current medications, and monitoring needs. This project helps owners organize questions; it does not provide diagnosis or dosing guidance.
